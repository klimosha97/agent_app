СПРАВОЧНИК МЕТРИК ПО ПОЗИЦИЯМ (Rustat)
=====================================

Назначение:
- Для каждого игрока берём его позицию (position_code)
- Сравниваем его статистику ЗА ТУР (по сути на 90 минут) с средними PER90 по турниру
- Используем только релевантные метрики для его позиции

Формат:
- core: самые важные метрики для оценки роли
- support: дополнительные метрики профиля
- risk: дисциплина/ошибки (штрафные)

Примечание:
- Метрики указаны как metric_code из системы (METRICS_MAPPING)
